# Quality Gates

Mandatory checks that run before ANY output reaches the user. These are not suggestions.

## Quick Checks (all must pass)

### Tension Check
- [ ] Does the strategy create a felt problem before offering a solution?
- [ ] Is there a named tension between two forces?
- [ ] Would a reader feel the problem viscerally, not just understand it intellectually?
- [ ] If you removed the solution, would the problem alone be compelling?

### Specificity Check
- [ ] Is every insight specific to THIS brand in THIS cultural moment?
- [ ] Could you swap in a different brand name and have it still work? If yes, rewrite.
- [ ] Are cultural references named and dated (a specific tweet, trend, behavior, data point)?
- [ ] Are audience descriptions based on behavior and belief, not just demographics?

### Argument Check
- [ ] Does each section build on the previous one, or could they be rearranged?
- [ ] Is there a single clear line from problem to insight to strategy to expression?
- [ ] Does the strategy feel earned by the argument, or bolted on?
- [ ] Is the coined concept a genuine compression of the insight, or a label stuck on top?

### Consultant-Speak Check
- [ ] Any sentence that sounds smart but says nothing? Kill it.
- [ ] Any framework used as a deliverable rather than a thinking tool? Simplify or cut.
- [ ] Any use of: "ecosystem," "360," "end-to-end," "holistic," "synergy," "paradigm," "stakeholder alignment"? Replace with plain language.
- [ ] Any sentence a CMO has heard 100 times before? Rewrite with the specific thing.

### Cultural Grounding Check
- [ ] Is the cultural truth actually true right now, or recycled?
- [ ] Would someone living inside the culture recognize this as their experience?
- [ ] Is the cultural reference doing strategic work (driving the insight), or decoration?

### Copy Quality Check
- [ ] Voice DNA applied (contractions, short paragraphs, physical verbs, no em dashes)?
- [ ] Stop-slop full checklist passes?
- [ ] No banned phrases from voice.md?

## Scoring Rubric (5 dimensions, 1-10 each)

| Dimension | What 10 looks like |
|-----------|-------------------|
| **Tension** | Reader feels stuck in the problem. Tension is specific, cultural, urgent. Removing the solution leaves a compelling provocation. |
| **Specificity** | Named cultural signals, real quotes, specific competitive gaps. No sentence could apply to a different brand. |
| **Narrative Arc** | Each section earns the next. Insight feels inevitable in retrospect but surprising when you reach it. Strategy is the only logical conclusion. |
| **Cultural Grounding** | Cultural references do strategic work. Strategy collapses without them. Someone inside the culture would nod, not roll their eyes. |
| **Persuasion** | Reader arrives at the strategy feeling like they discovered it themselves. Argument closes escape routes. Coined concept sticks after closing the doc. |

### Scoring Scale
- **8-10**: Exceptional. Real strategic work.
- **6-7**: Solid but could be sharper. One or two elements carrying weight, others filling space.
- **4-5**: Generic. Structure is right but content could apply to multiple brands.
- **1-3**: Consultant filler. Sounds professional but says nothing.

### Thresholds
- **Below 35/50**: Revise internally. Do not present to the user.
- **35-40**: Present with flagged weak dimensions.
- **Above 40**: Present.
- **Any dimension below 7**: Flag with the specific section and what's weak.

## Anti-Patterns (hard failures)

Any of these appearing in output is a failure. Fix before presenting.

1. **The Generic Cultural Take** — "Gen Z values authenticity." Could appear in any deck. Fix: name the specific signal.
2. **Strategy Without Tension** — Solution arrives before the argument earns it. Fix: spend more time on the problem.
3. **Consultant-Speak** — "360 omnichannel experience ecosystem." Fix: say the specific thing.
4. **The Insight That Isn't** — Observations presented as insights. Fix: find the tension within the observation.
5. **Strategy by Adjective** — "Bold, authentic, innovative." Fix: replace adjectives with decisions.
6. **The False Bridge** — "From product to experience" without earning the bridge. Fix: trace logic from preceding sections.
7. **Mood Board as Strategy** — Aesthetic direction without strategic grounding. Fix: connect every visual reference to a decision.
8. **The Recycled Framework** — Framework as deliverable instead of thinking tool. Fix: use it to think, then discard.

## Mode-Specific Gates

Different modes have additional checks beyond the universal gates:

- **Narrative**: Slide copy gets its own line-by-line Voice DNA + Stop-Slop pass (per feedback-log.md 2026-03-18)
- **Brief**: Proposition must be a single sentence that forecloses alternatives
- **Product**: Success metrics must be measurable, not aspirational
- **Courtroom**: Synthesizer output must trace which objections survived and which didn't
- **Pressure Test**: Every critique must suggest a specific fix, not just name the problem
